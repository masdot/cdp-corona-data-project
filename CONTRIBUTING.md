# Contributing to CDP

**Welcome!** Thank you for your interest in contributing to the corona data project!


## Reporting Issues
If you find any problems in the code or if you have problems using it, you can report this by opening an issue on Github. Please check first if the issue is already open or maybe already solved or in progress. If the issue already exists, please add a reaction (ğŸ‘).


## Requesting or adding an feature
If you request a feature or are interested in adding a feature, please describe clearly what it is about.
